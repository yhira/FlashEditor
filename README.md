# FlashEditor

FlashEditorは、Windows向けの軽量で低機能なメモアプリケーションです。.NET 9.0 (Windows Forms) で構築されており、シンプルながらも「あると便利」な機能を備えています。

## 主な機能

- **常に手前に表示 (TopMost)**: ピン留めアイコンでウィンドウを常に最前面に固定できます。
- **履歴管理 (Undo/Redo)**: 自動スナップショット機能により、安心して編集操作を行えます。
  - 3秒間操作がない場合に自動で履歴を保存します。
  - カスタム描画された矢印アイコンで直感的に操作可能。
- **Google検索連携**: 選択したテキスト（未選択時は全文）をワンクリックでGoogle検索できます。
- **日時挿入**: 「新しいメモ」ボタンで、現在の日時と区切り線を瞬時に挿入。日報や作業ログに最適です。
- **テーマ対応**: システム設定に合わせて、ダークモードとライトモードを自動で切り替えます。
- **カスタムUI**: ベクターライクな描画コードによって生成された、スケーラブルで美しいアイコンを使用しています。

## 動作環境

- Windows OS
- .NET 9.0 Runtime

## 開発環境

- Visual Studio 2022
- .NET SDK 9.0

## インストールと実行

1.  このリポジトリをクローンします。
    ```bash
    git clone https://github.com/your-username/FlashEditor.git
    ```
2.  Visual Studio 2022 で `FlashEditor.sln` を開きます。
3.  ビルドして実行します。
