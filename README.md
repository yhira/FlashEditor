# FlashEditor

FlashEditorは、Windows向けの軽量で直感的なメモアプリケーションです。.NET 9.0 (Windows Forms) で構築されており、シンプルで高速な動作に加え、「あると便利」な機能を多数備えています。思考を妨げないクリーンなUI設計と、作業効率を最大化する各種機能により、日々のメモ取りやタスク管理を強力にサポートします。

## 🌟 主な機能

### 📝 快適なテキスト編集と履歴管理

- **高度な履歴管理 (Undo/Redo)**: 独自のUndo/Redo機能により、文字入力や行の増減などの変化を自動で検知し、履歴（スナップショット）を保存（最大1000件）。操作ミスがあっても確実に過去の状態へ復元でき、長時間の編集作業でも安心です。
- **日時と区切り線のワンクリック挿入**: 「新しいメモ」ボタンで、現在の日時と区切り線を瞬時に追加。日報やタイムスタンプ付きの作業記録を素早く作成できます。
- **コンテンツの自動保存**: アプリ終了時に内容を自動保存し、次回起動時にそのまま復元します。

### 🎨 洗練されたカスタマイズ可能なUI

- **テーマ対応 (ライト / ダークモード)**: OSのシステム設定に連動するほか、手動での切り替えにも対応。目に優しいダークモードでは、タイトルバーからコンボボックス、ツールチップに至るまでUIの細部までテーマが美しく適用されます。
- **モダンでスケーラブルなフラットアイコン**: どんなサイズに拡大しても劣化しない、プログラムベースでカスタム描画されたマルチサイズ対応のモノラインアイコンを採用。モダンでクリーンな操作感を実現しました。
- **ツールボタンサイズ調整**: ユーザーの好みに合わせて、ツールバーのアイコンサイズを3段階（Small / Medium / Large）で変更可能です。
- **フォントの自由な変更**: お好みのフォントとサイズを自由に設定可能です。

### 🌐 多言語対応 (i18n)

- **グローバルサポート**: 外部JSONファイルを用いた言語システムを採用。日本語、英語をはじめ、中国語（簡体字/繁体字）、韓国語、フランス語、スペイン語、ポルトガル語、イタリア語、ドイツ語に完全対応。OSの言語設定に合わせて自動でUIの言語が切り替わります（手動設定も可能）。

### ⚡ 便利なユーティリティ機能

- **常に手前に表示 (TopMost)**: ピン留めアイコンをクリックするだけで、ウィンドウを他の最前面に固定。ブラウザや資料を見ながらのメモ取りがスムーズになります。
- **Google検索連携**: 選択したテキストをワンクリックで既定のブラウザからGoogle検索。調べ物をしながらの作業効率が飛躍的に向上します。
- **完全ポータブル設計**: インストーラー不要。配布用ZIPファイルを展開するだけで動作し、設定ファイルや履歴も同じフォルダに保存されるため、USBメモリ等に入れて持ち運ぶ環境としても最適です。

## 📦 配布ファイルの構成 (FlashEditor.zip)

リリース等で配布される `FlashEditor.zip` は以下の構成になっています。展開して `FlashEditor.exe` を実行するだけで、すぐにご利用いただけます。

```text
FlashEditor/
├── FlashEditor.exe       (アプリケーション本体)
├── FlashEditor.dll       (依存ライブラリ)
├── FlashEditor.runtimeconfig.json (ランタイム構成)
└── lang/                 (多言語対応用フォルダ)
    ├── en.json           (英語)
    ├── ja.json           (日本語)
    ├── zh-CN.json        (簡体字中国語)
    └── ...               (その他の言語ファイル)
```

## 💻 動作環境

- Windows 10 / 11
- .NET 9.0 Runtime

## 🛠️ 開発環境

- Visual Studio 2022
- .NET SDK 9.0

## 🚀 インストール・ビルド方法

1.  ソースコードからビルドする場合は、このリポジトリをクローンします。
    ```bash
    git clone https://github.com/your-username/FlashEditor.git
    ```
2.  Visual Studio 2022 で `FlashEditor.sln` を開きます。
3.  ビルド（F5 または Ctrl+Shift+B）して実行します。

※ ビルド済みバイナリを利用する場合は、配布ZIPを展開し `FlashEditor.exe` を実行するだけでお使いいただけます。
