================================================================================
  FlashEditor
================================================================================

FlashEditor は、Windows向けの軽量で直感的なメモアプリケーションです。
シンプルで高速な動作に加え、「あると便利」な機能を多数備えています。

--------------------------------------------------------------------------------
 💻 動作環境
--------------------------------------------------------------------------------

- Windows 10 / 11
- .NET 9.0 Runtime

※ アプリケーションが起動しない場合は、マイクロソフトの公式サイトより
　 「.NET 9.0 Desktop Runtime」をインストールしてください。

--------------------------------------------------------------------------------
 🚀 使い方とファイル構成
--------------------------------------------------------------------------------

[インストールと起動]
本アプリケーションはポータブル設計のためインストーラーはありません。
フォルダ内にある「FlashEditor.exe」をダブルクリックするだけで、
すぐに起動・利用できます。
USBメモリなどに入れて別のパソコンへ持ち運ぶことも可能です。

[詳しい使い方について]
画面構成や操作方法、設定のカスタマイズ（テーマやフォント変更など）について
は、同梱されている `manual.md` をご覧ください。
※ .md ファイルは通常のテキストファイルになりますので、
　 Windowsのメモ帳などのテキストエディタでご覧いただけます。

[配布ファイルの構成]
FlashEditor/
├── FlashEditor.exe                (アプリケーション本体・ここから起動)
├── FlashEditor.dll                (プログラムの実体・依存ライブラリ)
├── FlashEditor.runtimeconfig.json (動作環境の設定ファイル)
├── FlashEditor.deps.json          (動作に必要な部品リスト・依存関係ファイル)
├── Readme.txt                     (このファイル)
├── manual.md                      (操作マニュアル)
└── lang/                          (多言語対応用フォルダ)
    ├── en.json                    (英語)
    ├── ja.json                    (日本語)
    ├── zh-CN.json                 (簡体字中国語)
    └── ...
